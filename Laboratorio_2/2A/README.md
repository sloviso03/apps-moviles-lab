# Lab 2A — Lista de Películas (Modelo Tradicional)

Proyecto base para el Lab 2A de la materia Aplicaciones Móviles.

## Cómo abrir el proyecto

1. Abrir Android Studio
2. `File → Open → seleccionar esta carpeta`
3. Esperar a que Gradle sincronice las dependencias
4. Ejecutar con `Shift + F10` o el botón Run

## Archivos para completar

| Archivo | Etapa |
|---|---|
| `res/layout/item_pelicula.xml` | Etapa 1 |
| `ui/peliculas/PeliculasViewModel.kt` | Etapa 2 |
| `ui/peliculas/PeliculasFragment.kt` | Etapa 3 |

## Archivos de referencia (completos)

- `res/layout/fragment_detalle.xml` — ejemplo terminado de layout XML
